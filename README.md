# D2 GRUP — Kurumsal Web Sitesi Tasarımları

Güzellik ve estetik merkezlerine ileri teknoloji cihazlar sunan **D2 Grup** için hazırlanan iki farklı frontend tasarım konsepti. Tamamen statik (HTML + CSS + JS), harici bağımlılık yok — çevrimdışı çalışır.

## Klasörler

| Klasör | Tema | Açıklama |
|--------|------|----------|
| [`d2grup-dark/`](d2grup-dark/) | Koyu / Lüks | Siyah + altın, premium hava |
| [`d2grup-light/`](d2grup-light/) | Açık / Minimalist | Beyaz + siyah, editoryal minimal |

## Sayfalar (her iki temada)

- **index.html** — Anasayfa
- **kurumsal.html** — Kurumsal / Hakkımızda
- **kategoriler.html** — Yüz / Vücut / Longevity kategorileri
- **teknolojiler.html** — Teknolojiler (Lazer, RF, HIFU, IPL, Ultrason, Soğutma, EMS, Akustik Dalga)
- **markalar.html** — Partner markalar
- **kozmetik.html** — Kozmetik ürünler
- **urunler.html** — Ürün kataloğu (filtreli)
- **urun-detay.html** — Ürün detay sayfası
- **iletisim.html** — İletişim
- **bayi-girisi.html** — Bayi girişi

## Kullanım

Herhangi bir tarayıcıda ilgili klasördeki `index.html` dosyasını açmanız yeterlidir.

## Notlar

- Cihaz ve ürün görselleri şu an CSS/SVG yer tutuculardır; gerçek ürün fotoğraflarıyla değiştirilebilir.
- Kurulum veya build gerektirmez.
